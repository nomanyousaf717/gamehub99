🔢 How to Play Sudoku Pro

Objective: Fill the 9×9 grid so that every row, column, and 3×3 box contains digits 1-9!


Controls:

  • 🖱️ Click a cell to select it
  • 🔢 Click a number or use keyboard 1-9
  • 📝 Use notes mode for possible numbers
  • 🗑️ Erase button or Delete key to clear cells

Difficulty Levels:

  • 🟢 Easy: 36-46 given numbers
  • 🟡 Medium: 28-35 given numbers
  • 🔴 Hard: 22-27 given numbers
  • 🟣 Expert: 17-21 given numbers

Scoring:

  • 🎯 Points based on difficulty and time
  • ⚡ Time bonus for quick completion
  • ❌ Penalty for mistakes (3 mistakes = game over)
  • 💡 Hint penalty but helps when stuck
